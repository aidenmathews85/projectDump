Hi there!

This repository serves as a showcase of some of my projects.

The "Projects" subfolder contains past, basic Unity games that I have worked on.

The subfolder titled "BetterBossRush" is a mod for the game Terraria and is an addon for another mod, Calamity, that also modifies Terraria. I am most proud of this project, as this was one of the first projects that required me to learn a whole new syntax (tModLoader). Also, the code is easily accessible in the BetterBossRush.cs file.

The mod basically modifies an event in Calamity known as the Boss Rush, where players must fight every boss in the game in sequential order. My mod greatly speeds this event up by rewriting certain parts of the Boss Rush itself to spawn more bosses at once in earlier tiers of the event.
